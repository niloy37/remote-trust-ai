# Demo Script

## 30-Second Setup

RemoteTrust AI helps global job seekers avoid fake, restricted, or low-quality remote jobs before they spend hours applying.

## Walkthrough

1. Open the landing page.
2. Click **Analyze a Job**.
3. Click **Try Verified Example**.
4. Run analysis and point out:
   - final trust score
   - Verified badge
   - high legitimacy, remote authenticity, eligibility, and quality scores
   - positive signals like salary, official hiring link, remote-first policy, and interview process
5. Click **Try Scam Example**.
6. Run analysis and point out:
   - Risky verdict
   - payment requests
   - Gmail or chat-app contact
   - no-interview urgency
7. Click **Try Restricted Remote Example**.
8. Show how a legitimate job can still be a poor match for a global applicant because it is US-only.
9. Open the dashboard.
10. Filter by verdict and applicant country.
11. Open a saved result and submit feedback.

## Closing

The MVP combines practical rule-based AI, NLP-style feature extraction, structured scoring, local persistence, and a feedback loop. It is demo-ready today and designed to grow into browser extensions, ATS verification, company reputation checks, and fine-tuned models.
